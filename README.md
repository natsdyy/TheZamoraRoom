# The Zamora Room

A premium, minimalist industrial-luxury web application for **The Zamora Room — Coffee Lab**.

## Features
- **Dynamic Menu**: Real-time management of coffee and non-coffee selections.
- **Merchandise Store**: Showcase of curated goods.
- **Admin Dashboard**: Comprehensive control over site content and assets.
- **Modern UI**: Built with React, TypeScript, and Framer Motion for a stunning user experience.

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Zustand (State Management)

## Getting Started
1. `npm install`
2. `npm run dev`
